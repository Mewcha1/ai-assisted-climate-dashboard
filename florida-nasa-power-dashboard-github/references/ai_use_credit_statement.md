# AI Use and Credit Statement

This capstone project was developed with ChatGPT AI assistance. The student provided repeated prompts, feedback, corrections, and design direction throughout the process. ChatGPT generated or assisted with the dashboard code, data download code, report drafting, README documentation, prompt appendix, executive summary, and packaging of files.

The student role was primarily to guide the project through iterative prompting, review outputs, identify issues, request corrections, and approve dashboard design changes. AI-generated outputs were reviewed and revised through repeated feedback to improve correctness, clarity, and alignment with the project instructions.
